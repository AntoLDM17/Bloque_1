# git_leaks
## Leak finder in git commits.
### This ETL will explore every commit in the branch of a repository and try to find some key words. It prints the leaks and where did it found them. 
## Instructions: 
### Start by cloning the repo and change the repo folder location in the ".py". Now, type the branch you want to search leaks in in the ".py". Use pip install -r requirements.txt to download the libraries needed. Download the repo skale from this link "https://github.com/skalenetwork/skale-manager". ยง  skale-manager (you get this by first creating the skale folder and the cloning the repo you choose)
